# 2026 The International: Strategic Roadmap

이 프로젝트는 2026년 전략 로드맵을 위한 인터랙티브 프레젠테이션 애플리케이션입니다. React와 Tailwind CSS를 사용하여 제작되었으며, 시각적으로 매력적인 전환 효과와 데이터 시각화를 제공합니다.

## 프로젝트 구조

```
src/
  components/
    common/       # 공통 UI 컴포넌트 (SlideWrapper, SectionHeader 등)
    slides/       # 각 슬라이드별 컴포넌트 (01_Intro.tsx ~ 25_MissionReady.tsx)
constants.tsx     # 데이터 상수
App.tsx           # 메인 애플리케이션 (상태 관리 및 네비게이션)
```

## 시작하기

### 필수 조건
- Node.js

### 설치

```bash
npm install
```

### 실행

```bash
npm run dev
```

## 조작 방법

- **이전 슬라이드**: 왼쪽 방향키 (`←`) 또는 화면 왼쪽 하단 버튼
- **다음 슬라이드**: 오른쪽 방향키 (`→`), 스페이스바 (`Space`) 또는 화면 오른쪽 하단 버튼

## 주요 기능

- **시네마틱 전환**: CSS 애니메이션을 활용한 부드러운 슬라이드 전환
- **데이터 시각화**: 차트 및 지표의 시각적 표현
- **키보드 네비게이션**: 프레젠테이션 편의를 위한 키보드 조작 지원
